# 前後端分離專案，使用技術如下
> vuejs vue-router vuex muse-ui axios express knex backpack restful ......

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# server dev localhost:8099
npm run server

# webpack打包至'dist'目錄，含node server會包成index.js
npm run build

```
